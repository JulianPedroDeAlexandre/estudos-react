# Gerador de Senhas em React

Este projeto é um **gerador de senhas aleatórias** desenvolvido em **React** utilizando **Vite** como bundler.  
Ele permite criar senhas seguras e copiá-las facilmente para a área de transferência.

---

## Tecnologias utilizadas
- [React](https://react.dev/) – Biblioteca para construção de interfaces
- [Vite](https://vitejs.dev/) – Ferramenta de build rápida para projetos front-end
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) – Linguagem principal
